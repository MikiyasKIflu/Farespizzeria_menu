description: Create a new café instance from the master template
---

// turbo-all
1. Ensure you are in the project root directory (`c:\Users\Christmas\Desktop\cafe-menu-template`).
2. Run the initialization script:
```bash
npm run setup-cafe
```
3. Follow the interactive prompts to set:
   - Café Name
   - Slogan
   - WhatsApp Number
   - Branding Colors (Primary & Accent)
4. Update the `.env` file with the new café's Supabase credentials.
5. Replace `public/logo.png` with the new café's logo.
6. Run the development server to verify changes:
```bash
npm run dev
```
